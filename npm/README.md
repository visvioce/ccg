# CCG

CCR Gateway (Go) - A Go rewrite based on [CCR](https://github.com/musistudio/claude-code-router)

## Installation

```bash
npm install -g @visvioce/ccg
```

## Usage

```bash
# Start server
ccg start

# Stop server
ccg stop

# Check status
ccg status

# Open terminal UI
ccg tui
```
